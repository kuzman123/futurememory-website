# Future Memory

Design and strategy studio website.

- **Live:** https://futurememory.studio
- **Built with:** Vanilla HTML, CSS, JavaScript
- **No backend, no login, no build step**

## Local development

```bash
python -m http.server 8000
open http://localhost:8000
```

Edit index.html, save, refresh.

## Deploy

Drag folder to [netlify.com/drop](https://netlify.com/drop) or push to GitHub for Vercel auto-deploy.
